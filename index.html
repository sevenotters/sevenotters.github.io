The Seven Otters Project

A CQRS/ES library and facilities for Elixir to build events oriented applications.

Learn more

What is Seven Otters?

Seven Otters is a set of facilities (macroes, functions, modules, etc.) developed to create CQRS/ES solutions in Elixir on BEAM virtual machine.
With Seven Otters you can easly define events, aggegates, aggregate roots, commands, projections, service, policies, and batches. Moreover Seven Otters internally implements a command bus and an events (pub/sub) bus.
Seven Otters is published as open source project.

What'a about the name?

Seven Otters is simply the anagram of events store.

Seven Otters is in development phase, although it is already used in various commercial projects.
